## 🐛 Häufige Probleme

### Problem: "Popup wurde blockiert"
```
Lösung:
1. Browser-Einstellungen öffnen
2. Popups für bondageprojects.elementfx.com erlauben
3. BC neu laden
4. Nochmal versuchen
```

### Problem: "Spieler nicht gefunden"
```
Lösung:
1. Stelle sicher, dass du in einem Raum bist
2. Klicke "🔄 Aktualisieren"
3. Wenn weiterhin Problem: BC neu laden
```

### Problem: "Code wird nicht ausgeführt"
```
Lösung:
1. F12 drücken → Console öffnen
2. Fehler suchen
3. Meist: Fehlende Berechtigungen oder Item blockiert
```

### Problem: "Items erscheinen nicht"
```
Mögliche Ursachen:
- Fehlende Berechtigungen
- Falscher Body-Typ (Male/Female)
- Item wird durch anderes Item blockiert
- Slot ist bereits belegt
```

## 🎓 Für Fortgeschrittene

### Custom Domains
```
Du kannst eine eigene Domain verwenden:
1. Domain kaufen
2. CNAME-Record auf GitHub Pages setzen
3. In Repository-Settings konfigurieren
```

### CDN verwenden
```
Statt Raw-URL, jsDelivr verwenden:
https://cdn.jsdelivr.net/gh/USER/REPO@main/FILE.js

Vorteile:
✓ Schneller
✓ Besseres Caching
✓ Globales CDN
```

### Script minifizieren
```
Für noch kleinere Bookmarklets:
1. JavaScript Minifier verwenden
2. Variablen verkürzen
3. Kommentare entfernen
```
