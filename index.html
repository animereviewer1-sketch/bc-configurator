Ziehe diesen Button in deine Lesezeichen-Leiste:

🚀 BC Configurator

Hinweis: Ersetze die URL mit deinem GitHub-Repository!

📝Anleitung

  1. Lesezeichen-Leiste einblenden (in den meisten Browsern: Strg+Shift+B)
  2. Ziehe den Button oben in deine Lesezeichen-Leiste
  3. Öffne Bondage Club und gehe in einen Raum
  4. Klicke das Bookmarklet in deiner Lesezeichen-Leiste
  5. Fertig! Der Konfigurator öffnet sich in einem neuen Fenster

🎯Automatische Spieler-Erkennung

Findet automatisch alle Spieler im aktuellen Raum. Keine manuelle Eingabe mehr!

⚡Direkte Code-Ausführung

Code wird automatisch ausgeführt - kein Kopieren und Einfügen mehr nötig!

👥Multi-Target Support

Wende Items auf dich selbst oder andere Spieler im Raum an.

🎨Vollständige Konfiguration

Farben, Module, Optionen, Schlösser - alles konfigurierbar!

📦Outfit-Management

Erstelle komplexe Outfits mit mehreren Items für verschiedene Spieler.

🪟Separates Fenster

Übersichtliche UI in einem eigenen Fenster - BC bleibt sichtbar!

⚠️Wichtige Hinweise

Popup-Blocker: Stelle sicher, dass Popups für Bondage Club erlaubt sind!

Berechtigungen: Du kannst nur auf Spieler zugreifen, die dir entsprechende Rechte gegeben haben.

GitHub Setup: Vergiss nicht, die URL im Bookmarklet mit deinem GitHub-Repository zu ersetzen!

🔧Für Entwickler: GitHub Setup

  1. Erstelle ein neues GitHub Repository (z.B. "bc-universal-configurator")
  2. Lade die Datei bc-universal-bookmarklet.js hoch
  3. Aktiviere GitHub Pages in den Repository-Einstellungen
  4. Ersetze in dieser HTML-Datei die URL mit deiner Raw-GitHub-URL
  5. Die Raw-URL findest du, wenn du die .js Datei öffnest und auf "Raw" klickst

💡Quick Start

Alternative für Tests: Wenn du das Script testen möchtest, ohne GitHub zu verwenden, kannst du den Inhalt von bc-universal-bookmarklet.js auch direkt als Bookmarklet-URL verwenden. Füge einfach javascript: davor und entferne alle Zeilenumbrüche.
